# Team Synergy Showcase

Immersive, high-fidelity website showcasing SAM, Jenkins, Rick, and Doc Oc's combined expertise in Maps and Search technologies.

## ✨ Design System

**Color Palette:**
- `--color-primary`: `#007bff` (Electric blue—connectivity & energy)
- `--color-secondary`: `#ffc107` (Gold accent for highlights)
- `--color-background-base`: `rgba(20, 25, 40, 0.8)` (Deep indigo with transparency)
- `--color-surface`: `rgba(255, 255, 255, 0.08)` (Glass effect base)
- `--color-text-primary`: `#ffffff` (Clean white)
- `--color-text-secondary`: `rgba(255, 255, 255, 0.7)` (Supporting text)

**Typography:**
- Headers: Inter (medium to semi-bold)
- Body: Roboto

**Effects:** Glassmorphism with 10px backdrop blur and thin border highlights.

## 🎯 Features

### Team Tabs Section
Horizontal tab navigation displaying all four team members with:
- Glowing circular avatars
- Bold name headers
- Role summaries as subtitles
- Specialty badges as pill-shaped chips

### What's Next Section
Hexagonal node-based project proposals focusing on Maps & Search, including:
- Visual team-member overlap diagrams
- Hover-to-expand interactions revealing 2-3 deliverable bullet points

### Responsive Design
- **Mobile** (<600px): Vertical stack with hamburger nav
- **Tablet** (601-1024px): 2-column grid for team cards
- **Desktop** (1025px+): 3-4 column glassmorphism showcase

### Animations
- Subtle entrance: Fade-in + slide-up, staggered on load
- Smooth 300ms hover transitions
- No jarring effects—all polish

## 🚀 Project Proposals

1. **Geo-Search Integration Hub** – Vector search + maps team
2. **Spatial Knowledge Graph** – Entity relationship mapping
3. **AI Route Optimizer** – ML-powered route planning
4. **Semantic Map Explorer** – NLP-driven map interface
5. **GeoVis Analytics Dashboard** – Visualization analytics
